ReadMe.txt
===========

Diese zip file im Projektverzeichnis eines neu angelegten, dynamischen Web
Projekts (DWP) entpacken. Nicht vergessen, unter "Preferences" die Checkbox bei 
"Project Facets" zu setzten.
Es werden folgende Dateien und Verzeichnisse angelegt:

Projektverzeichnis
      |
      |--- src
      |    |--- net.lehre_online.pf_vorlage
      |                             |--- MyBean.java
      |--- WebContent
             |--- META-INF 
             |       |--- context.xml
             |
             |--- index.html
             |--- vorlage_pf.xhtml
             |--- WEB-INF
             |       |--- beans.xml 
             |       |--- faces-config.xml
             |       |--- web.xml 
             |
             |--- resources
                     |--- css
                     |     |--- myCss.css
                     |
                     |--- etc
                     |--- vorlage_pf.zip
                     |--- Diese ReadMe-Datei
                     |
                     |--- img
                           |--- prime_faces.png 
                           |--- java.png 
             
 Nach dem Entpacken werden die Dateien durch ein Refresh (F5) in der IDE 
 angezeigt.
